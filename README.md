# BlockedDomainsFilter
A uBlock Origin filter list to block 'fake' domains from google search results.

## How to use
1) From ```uBlockOrigin``` settings go to ```Filter lists```  
2)  Scroll down to ```Import``` and copy the filter Raw URL
